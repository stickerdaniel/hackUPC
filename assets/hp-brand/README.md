# HP Brand Assets

Mirror of HP's official brand assets pulled from [HP Brand Central](https://www.hp.com/us-en/hp-information/brandcentral/resource-library.html). Every ZIP in `zips/` is the unmodified file served by `https://www.hp.com/content/dam/sites/brand-central/private/<name>.zip` (HP login required).

Some packages contain assets larger than 100 MB; the directory is tracked with Git LFS via `.gitattributes`.
