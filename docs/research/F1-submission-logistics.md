### Submission format — what Devpost requires (standard HackUPC rules)
- One Devpost project per team (max 4 members).
- Devpost username must resemble your real name.
- You must explicitly **opt in** to each sponsor prize (including HP) inside the
  Devpost submission form; opting in is not automatic.
- A GitHub repo link is expected as part of the project description (no specific
  mandate in public rules, but standard practice at HackUPC).

### Judging format
- **Expo style** — judges rotate to each team's table.
- **3 minutes per demo** per judging room.
- **Live code demo is mandatory** — slides and video are not allowed as a substitute;
  using them "severely undermines" evaluation.
- Sponsor-track participants attend **two separate expo sessions**: (1) the HP
  judging room (sponsor), and (2) a general HackUPC judging room.
- Judging criteria: Technology · Idea & Innovation · Learning.
